Balbino

Balbino is my homemade game engine. I made this during my second year at Digital Arts and Entertainment. I decided to overhaul it and keep working on it.

The engine is written in C++23 and uses Assimp, EnTT, GLI, GLM, mINI, SDL2, SDL2 images, Vulkan SDK, ImGui... and GitHub as source control.

Child of Lothian

This was a group project of about 22 students, in which I was the AI programmer. In this project, we use Unreal Engine 4 C++ for the heavy lifting and blueprinting to create connections. We used Perforce and Swarm to sinc our work and to make pull requests.

AI programming

In this program, we had a character that had to survive as long as possible in a zombie apocalypse. We programmed the AI to run from house to house and see if an item lies within it. If the AI found an item, it would pick it up, validate how useful it is and take action accordingly.

In this project, I used C++17.

Flemish programming contest

In this contest, we had to be in a team of three. You got five problems, and you had to solve them using programming. The twist is that only one screen, keyboard and mouse were allowed. Only one person could do the programming. The other two had to figure out how to solve the problem and tell the typing programmer what to do.

Robocontest at VIVES

The goal was to follow a white line with a small autonomous car via one or more optical sensor(s).

Because of some mechanical problems, we had to rewrite the entire program one minute before the start of the competition.

Balbino Engine

Explanation

This Game Engine was originaly used in a school assignment. Balbino has now been revamped multiple times and is starting to become a hobie and a project to show off to sompanies

The used Lybraries are:

  • Open Asset Import Library(Assimp), for loading 3d models;
  • EnTT as an entity component system;
  • OpenGL Image (GLI), for reading in DDS files;
  • OpenGL Mathematics (GLM), for math;
  • mINI for reading and writing ini files;
  • SDL2, to initiate the window and handel input events;
  • SDL2_images, for loading in images;
  • Vulkan, as render API.
  • ImGui, ImGuizmo and imNodes for the UI.
  • HDRI loader from Igor Kravtchenko, to load in HDRI's;
  • mikktspace, to correctly load tangents;

Some of the current features include:

  • Mesh Importer
  • UI
  • Shader Editor
  • Lighting
  • Material Editor
  • Custom (Binalry)File type to unify all assets
